  
## Configs
- System: Ubuntu 24.04.3 ADM64.

## Installation
- Install tool with bash.
- Attention: Execute as root/admin. (chmod +x script.sh)

```bash
  cd /opt/ && git clone https://github.com/0xPetrvs/ConfigTools.git
  cd ToolsFile
  chmod +x tools.sh
 ./tools.sh
```
## Tools

#### Git, make, Gcc

```bash
apt-get --assume-yes install git make gcc
```
#### Net Tools

```bash
apt install net-tools
```
