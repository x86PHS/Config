# Configuration Files

- Este repositório tem como intuito armazenar arquivos de configuração das minhas máquinas com distro linux.


## Installation
- Install tool with bash.
- Attention: Execute as root/admin. (chmod +x script.sh)

```bash
  cd /opt/ && git clone https://github.com/0xPetrvs/ConfigTools.git
  cd ToolsFile
  chmod +x tools.sh
 ./tools.sh
```
